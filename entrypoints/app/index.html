Install language data

Installs the lemma dictionary and frequency list for the selected language (built from PoliMorf / lemmatization-lists / OpenSubtitles). Polish, German and English ship with the extension; other languages are downloaded. Without it, word forms are tracked literally and inflections of the same word count separately.

Install from local files instead

Pick the two TSV artifacts built by scripts/build-lang-data.mjs.

Import vocabulary

Auto-detects the format: the Lute database (lute.db, read directly), a Lute term CSV export, an Anki "Notes in Plain Text" export, or the CSV from manga-translator / language-reactor-clone. Statuses are mapped: 1–5 → learning stages, W / Well Known → known, I → ignored. Words are lemmatized on import; existing entries are never downgraded; multi-word terms are skipped.

Estimate your vocabulary

You'll see ~35 words sampled across the frequency spectrum. Answer honestly whether you know each one; the estimate marks the most frequent N words as known. Requires installed language data.

Result

You know roughly the top words.

Adjust if the estimate feels off, then apply.
